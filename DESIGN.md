# CogniTeck - Landing Page Layout & AI Generation Guide
Este documento contiene la estructura, copys (textos) y la estrategia visual diseñada para que la copies y pegues directamente en **Stitch (withgoogle.com)**. Está optimizada para que la IA entienda el enfoque de software, potencie tu servicio actual y genere expectativa para tu futuro SaaS.
---
## 🚀 1. Prompt Maestro para Stitch (withgoogle.com)
*Copia y pega este texto en el input principal de Stitch para generar la estructura base:*
> "Create a futuristic, premium, and highly technological landing page for a software company named 'CogniTeck'. The aesthetic must clearly scream 'software and advanced tech' at first glance. Use a dark mode theme (deep tech obsidian black/dark grey), neon cyan or electric blue accents for primary actions, and clean glowing border containers to simulate a modern Dashboard or SaaS interface. The structure should feel clean, high-converting, and development-focused, avoid generic corporate designs. Use standard fonts like Inter or Arial with a line spacing of 1.25 for absolute readability. Include 4 main sections: a hero section with a floating futuristic abstract code/UI widget on the right, an 'Our Ecosystem' section divided in two columns (Web Design & Development Services vs. Our Upcoming SaaS Innovation), a 'Future Roadmap' interactive timeline, and a tech footer."
---
## 📐 2. Estructura de Bloques y Contenido (HTML/Markdown Layout)
### SECCIÓN 1: HERO (Primer impacto visual)
*El objetivo aquí es dejar claro que CogniTeck es software y tecnología desde el segundo cero.*
*   **Estructura Visual:** 2 Columnas. Izquierda: Textos y CTAs. Derecha: Un widget interactivo abstracto que simule código o una interfaz de software flotante.
*   **Titular Principal (H1):** `Engineering the Digital Future` (o en español: `Ingeniería de Software para la Próxima Era Digital`)
*   **Subtexto:** `Construimos plataformas web de alto impacto e infraestructura tecnológica a medida. Desde interfaces de conversión hasta el desarrollo de nuestro próximo ecosistema SaaS inteligente.`
*   **CTAs (Botones):**
    *   *Botón Primario (Cian brillante):* `Iniciar un Proyecto Web`
    *   *Botón Secundario (Borde brillante/Transparente):* `Explorar nuestro Roadmap`
---
### SECCIÓN 2: NUESTROS PILARES (Servicio Actual + Futuro SaaS)
*Aquí dividimos la pantalla en dos tarjetas o contenedores con estilo "Dashboard Glassmorphism" para mostrar lo que haces hoy y hacia dónde vas.*
#### Tarjeta Izquierda: 🌐 Desarrollo y Diseño Web Premium (Servicio Disponible)
*   **Icono sugerido en Stitch:** Terminal de código o un navegador web.
*   **Título:** `Desarrollo Web & Landing Pages de Alta Conversión`
*   **Descripción:** No creamos plantillas básicas. Desarrollamos sitios web robustos, ultra-rápidos y optimizados para el crecimiento de tu negocio, aplicando los estándares modernos de la ingeniería de software.
*   **Detalles técnicos que resalta la IA:** `Código Limpio | Optimización SEO | Arquitectura Escalable`
#### Tarjeta Derecha: 🧠 CogniTeck Core (SaaS en Desarrollo / Próximamente)
*   **Icono sugerido en Stitch:** Un cerebro digital, un chip o un engranaje en movimiento constante.
*   **Título:** `Nuestro Próximo Hito: Software as a Service`
*   **Descripción:** Estamos incubando nuestra propia plataforma en la nube (SaaS). Una solución nativa digital diseñada para automatizar y optimizar flujos de trabajo complejos. Actualmente en fase alpha de desarrollo privado.
*   **Badge visual superior:** `[ EN DESARROLLO / ALPHA ]`
---
### SECCIÓN 3: ROADMAP TECNOLÓGICO
*Una línea de tiempo interactiva o una cuadrícula de fases para demostrar que, aunque están iniciando, tienen una visión de ingeniería clara.*
*   **Fase 01 - Cimientos (Actual):** `Lanzamiento de CogniTeck & Servicios Globales de Desarrollo Web.`
*   **Fase 02 - Infraestructura (Q3 2026):** `Arquitectura del Backend y Core de nuestro primer SaaS.`
*   **Fase 03 - Beta Cerrada (Q1 2027):** `Pruebas de estrés y despliegue del SaaS con partners estratégicos.`
---
### SECCIÓN 4: FOOTER TECNOLÓGICO
*   **Izquierda:** `CogniTeck` - *Building software that scales.*
*   **Derecha:** Links de navegación limpia (Servicios, Roadmap, Contacto).
*   **Fondo:** Líneas de cuadrícula muy tenues que simulen una matriz técnica.

## 🎨 3. Paleta de Colores y Estilos para Stitch
Si la IA te genera colores muy planos, puedes usar los comandos por bloque de Stitch para refinar la interfaz pidiéndole textualmente esto:
*   **Fondo principal:** `#0B0F19` (Un negro azulado muy profundo de estilo tecnológico).
*   **Texto principal:** `#F3F4F6` (Blanco grisáceo suave, nunca blanco puro para evitar fatiga visual).
*   **Color de acento/botones:** `#06B6D4` (Cian eléctrico/Cyberpunk).
*   **Bordes de tarjetas:** `#1E293B` con un sutil efecto de brillo (`box-shadow: 0 0 15px rgba(6, 182, 212, 0.15)`).
